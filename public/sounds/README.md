# 27-0 Sound Assets

Place lightweight `.mp3` or `.wav` files here. The game loads them automatically;
if a file is missing, synthesized fallback tones play instead (no crash).

Suggested files:

| File | Used for |
|------|----------|
| `click.mp3` | UI clicks, menu, tabs, filters |
| `select.mp3` | Player selected, position picked |
| `reveal.mp3` | Player choices revealed |
| `reroll.mp3` | Reroll used |
| `complete.mp3` | Squad position filled |
| `historic.mp3` | Historic player in offers |
| `legend.mp3` | Legend player in offers |
| `goat.mp3` | GOAT player in offers |
| `joe-mellor.mp3` | Joe Mellor easter egg mode |
| `super-sam-hallas.mp3` | Super Sam Hallas easter egg mode |
| `mode-normal.mp3` | Normal / Classic mode start |
| `mode-hard.mp3` | Hard mode start |
| `mode-draft.mp3` | Draft mode start |
| `mode-cup.mp3` | Legacy Challenge Cup tone (unused if `challenge-cup.mp3` present) |
| `challenge-cup.mp3` | Challenge Cup mode activation (tournament intro) |
| `hard-on.mp3` | Hard Mode enabled |
| `hard-off.mp3` | Hard Mode disabled |
| `season-start.mp3` | Season / cup simulation start |
| `win.mp3` | Narrow match win |
| `loss.mp3` | Match loss |
| `big-win.mp3` | Big / thrashing win |
| `upset.mp3` | Upset victory |
| `perfect.mp3` | 27-0 perfect season |
| `disaster.mp3` | 0-27 winless season |
| `crowd.mp3` | Season simulation complete |
| `trophy.mp3` | Cup final won |
| `cup-loss.mp3` | Cup final lost |
| `fail.mp3` | F grade / failure |
| `menu-open.mp3` | Sidebar menu open |
| `menu-close.mp3` | Sidebar menu close |
| `expand.mp3` | Panels expanded (match details, clubs, try scorers) |

Keep files short (&lt; 100 KB) and normalized volume for best results.
